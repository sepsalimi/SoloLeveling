# Future Roadmap

- Production audio upload, transcription, retry queue, and raw-audio deletion workflow.
- Rich review editors for category, social context, purpose tags, efficiency, mood, and energy.
- Previous-period comparison engine for analytics insights.
- CSV export in addition to JSON sharing.
- Full Supabase persistence for all activities, sessions, preferences, and voice notes.
- End-to-end tests with a simulator once native dependencies are installed.
- Offline pending voice note queue with background-safe retry only after the app is opened.
- Account deletion admin endpoint that removes the Supabase auth user after data deletion.
- Apple Health, Health Connect, smartwatch integrations, sleep tracking, food photography, location tracking, calendar integration, social sharing, friends, subscriptions, and passive monitoring remain out of scope for the MVP.
